TAAGC API Documentation

Complete API reference for TAAGC Global platform
https://taagc.website/api

📋 Overview

The TAAGC API provides programmatic access to manage users, investments, transactions, and more. All API requests should be made to the base URL above.

Authentication

Most endpoints require authentication via Bearer token:

Authorization: Bearer YOUR_FIREBASE_ID_TOKEN

Get your token by calling the login endpoint.

🔐 Authentication Endpoints

POST /auth?action=login

Authenticate user with email and password

https://taagc.website/api/auth?action=login
{
  "email": "user@example.com",
  "password": "yourpassword"
}
ResponseUser data + authentication token
POST /auth?action=register

Create a new user account

https://taagc.website/api/auth?action=register
{
  "email": "user@example.com",
  "password": "yourpassword",
  "fullName": "John Doe",
  "phone": "+1234567890",
  "role": "client"
}
GET /auth?action=me Auth Required

Get current authenticated user's information

https://taagc.website/api/auth?action=me
Headers: {
  "Authorization": "Bearer YOUR_TOKEN"
}
POST /auth?action=logout Auth Required

Logout user and revoke token

https://taagc.website/api/auth?action=logout
POST /auth?action=reset

Request password reset email

https://taagc.website/api/auth?action=reset
{
  "email": "user@example.com"
}

👥 Users Endpoints

Admin Required

GET /users?action=list

List all users (paginated)

https://taagc.website/api/users?action=list&limit=50
POST /users?action=create

Create a new user

https://taagc.website/api/users?action=create
{
  "email": "newuser@example.com",
  "password": "password123",
  "fullName": "Jane Smith",
  "role": "investor",
  "status": "active"
}
GET /users?action=get&userId={id}

Get specific user by ID

https://taagc.website/api/users?action=get&userId=USER_ID
PUT /users?action=update&userId={id}

Update user information

https://taagc.website/api/users?action=update&userId=USER_ID
DELETE /users?action=delete&userId={id} Super Admin

Delete a user

https://taagc.website/api/users?action=delete&userId=USER_ID
GET /users?action=stats

Get user statistics

https://taagc.website/api/users?action=stats
