TAAGC Global API Documentation

Base URL: https://taagc.website/api

Authentication

All endpoints require a Bearer token in the Authorization header:

Authorization: Bearer YOUR_FIREBASE_ID_TOKEN
GET /

API Information

Returns API status and available endpoints.

Auth Endpoints

POST /auth/login

Login

Authenticate user with email and password.

{
  "email": "user@example.com",
  "password": "yourpassword"
}

User Management

GET /users

List Users

Returns list of all users (admin only).

POST /users

Create User

Create a new user (admin only).

POST /set-role

Set User Role

Update user role (super admin only).

{
  "uid": "user_id_here",
  "role": "admin"
}

Investments

GET /investments

List Investments

Returns all investments.

POST /investments

Create Investment

Create a new investment (admin only).

Transactions

GET /transactions

List Transactions

Returns transaction history.

Events

GET /events

List Events

Returns upcoming and past events.

Reports

GET /reports

Generate Reports

Generate custom reports (admin only).
