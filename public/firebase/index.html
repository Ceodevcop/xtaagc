Solution 1: Use CORS Proxy

If Firebase is blocked by CORS, use a proxy server to forward requests.

Option A: Using cors-anywhere (Quick Test)

// Test with CORS proxy const PROXY_URL = 'https://cors-anywhere.herokuapp.com/'; const FIREBASE_URL = 'https://identitytoolkit.googleapis.com/v1/accounts:signUp?key=AIzaSyAI1MrcP977UgRLQoePxl64JOSyj9v4WpI'; fetch(PROXY_URL + FIREBASE_URL, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ email: 'ceo@taagc.website', password: 'Admin123!@#', returnSecureToken: true }) })

Option B: Your Own Proxy (PHP)

Create a proxy PHP file on your server:

<?php // firebase-proxy.php header('Access-Control-Allow-Origin: *'); header('Access-Control-Allow-Methods: POST, GET, OPTIONS'); header('Access-Control-Allow-Headers: Content-Type'); $url = 'https://identitytoolkit.googleapis.com/v1/accounts:signUp?key=' . $_GET['key']; $data = file_get_contents('php://input'); $ch = curl_init($url); curl_setopt($ch, CURLOPT_RETURNTRANSFER, true); curl_setopt($ch, CURLOPT_POST, true); curl_setopt($ch, CURLOPT_POSTFIELDS, $data); curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/json']); $response = curl_exec($ch); $httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE); curl_close($ch); http_response_code($httpCode); echo $response; ?>

Solution 2: JSONP Workaround

Use JSONP to bypass CORS restrictions (limited functionality).

<script> function handleResponse(data) { console.log('Auth response:', data); } const script = document.createElement('script'); script.src = 'https://identitytoolkit.googleapis.com/v1/accounts:signUp?key=AIzaSyAI1MrcP977UgRLQoePxl64JOSyj9v4WpI&callback=handleResponse'; document.head.appendChild(script); </script>

Solution 3: Create Node.js Backend

// server.js - Run this on your server const express = require('express'); const cors = require('cors'); const fetch = require('node-fetch'); const app = express(); app.use(cors()); app.use(express.json()); app.post('/create-admin', async (req, res) => { try { const response = await fetch( 'https://identitytoolkit.googleapis.com/v1/accounts:signUp?key=AIzaSyAI1MrcP977UgRLQoePxl64JOSyj9v4WpI', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(req.body) } ); const data = await response.json(); res.json(data); } catch (error) { res.status(500).json({ error: error.message }); } }); app.listen(3000, () => console.log('Proxy server running on port 3000'));
